Implementation

File: components/SectionThreeTestimonials.tsx

Export default with props:

export interface Testimonial {
  id: number;
  quote: string;
  author: string;
  photoSrc: string;    // default placeholder e.g. "/avatars/user1.jpg"
  photoAlt: string;
}

export interface SectionThreeTestimonialsProps {
  onLearnMore?: () => void;
  testimonials?: Testimonial[]; // default to the 5 above with placeholder images
}


Defaults:

photoSrc placeholders: /avatars/user1.jpg … /avatars/user5.jpg

Accessibility:

Scroller role="region" with aria-label="Testimonials carousel".

Each card has aria-roledescription="slide" and aria-label="Testimonial n of 5".

UX:

Horizontal container: overflow-x-auto, scroll-snap-type: x mandatory.

Cards: scroll-snap-align: start, rounded-2xl, shadow, p-4, bg-white (or bg-neutral-900 for dark), max width ~ 85% viewport.

Add gradient peeks left/right to hint scrolling.

Include a minimal demo page: app/demo/section-3/page.tsx rendering the component with default data.

No external UI libs beyond Tailwind.

Implementation

File: components/SectionFourAlbums.tsx

Export default with props:

export interface AlbumCard {
  id: number;
  title: string;
  description: string;
  imageSrc: string;   // default placeholder e.g. "/albums/album1.jpg"
  imageAlt: string;
}

export interface SectionFourAlbumsProps {
  onExploreAlbums?: () => void;
  albums?: AlbumCard[]; // default to the 3 cards above with placeholder images
}


Defaults:

imageSrc placeholders: /albums/album1.jpg, /albums/album2.jpg, /albums/album3.jpg

Accessibility:

Scroller role="region" with aria-label="Albums carousel".

Each card aria-roledescription="carousel item".

Buttons have aria-label and keyboard focus styles.

UX:

Container: overflow-x-auto, scroll-snap-type: x mandatory.

Card: scroll-snap-align: start, rounded-2xl, shadow, bg-white, overflow-hidden, w-[85%] on mobile.

Use edge fade gradients to hint scroll.

Include demo page: app/demo/section-4/page.tsx rendering with default data.

No external UI libs beyond Tailwind.
